## WoofWell

WoofWell is a frontend-only MVP for a dog care and wellness tracker. The project focuses on clean UI design, simple navigation, and user-friendly layouts for managing daily care routines and basic pet information.

## Tech Stack

* HTML

* CSS

* Bootstrap

* Vanilla JavaScript (minimal)

## Notes

This is a UI-focused project with no backend or data persistence. All features are for demo/UI only.
